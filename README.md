# electron-playground
Where the gang learns electron, react, and typescript

Setup
npm install webpack tslint-config-standard tslint-loader ts-loader tslint typescript -D
npm install babel-core babel-loader babel-preset-es2015-node babel-preset-react standard standard-loader -D
npm install electron html-webpack-plugin @types/electron @types/node -D
npm install react react-dom
npm install @types/react @types/react-dom -D
npm install jest jest-junit @types/jest -D
Using the demo from https://blog.scottlogic.com/2017/06/06/typescript-electron-webpack.html
